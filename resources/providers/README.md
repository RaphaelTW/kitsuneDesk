# Pacotes offline opcionais

Coloque aqui, antes do build, os diretórios `goanime`, `anime-cli-br`, `ani-cli` ou `fast-anime-vsr` preparados por quem gera o instalador. O conteúdo é incluído em `resources/providers` e fica disponível ao instalador interno antes de consultar a internet.

Binários de terceiros não são versionados por padrão. Quem montar um pacote deve conferir licença, origem, assinatura e SHA-256 de cada artefato.
